# 第十一章、貪愛與嗔恨

> 「唯一能擾亂平和的東西是欲望。」——萊斯特·利文森

---

## 如何對任一對象做貪愛與嗔恨的練習——擺脫你潛意識裡破壞性的想法與感受

也許你有這樣一個習慣，一件你希望能夠停止的事情；也許你生活中有一個處處干擾你的人，或者是令你煩惱的工作情況。以下是兩張空白的貪愛與嗔恨表格，你可以把它們印出來方便使用，至少保留一套空白的供將來使用。挑一個你想要進行練習的對象吧，我們將要分析自己對它的貪愛與嗔恨。當你準備好的時候，我們就開始吧，讀下一頁的表格。

---

### 貪愛——想要擁有的欲望（範例）

| 我對工作頭銜的貪愛 | 放下了嗎？ |
|---|---|
| 我貪愛自己工作上的頭銜。——想要被別人認同 | |
| 我喜歡當別人知道我多成功時對我的尊敬。 | |
| 我很努力地工作去贏得它，也將繼續努力工作去保持它。 | |

**練習指導：**

1. 寫下自己對某人事物的貪愛。
2. 寫下自己關於這貪愛的意識流動（就是讓你的思維圍繞這個自由活動）。
3. 無論哪種欲望被激發了，釋放它。
4. 寫下它給自己帶來的好處與壞處。
5. 想象自己永遠都不能擁有這個東西了，然後把這一想法激起的感受釋放掉。
6. 問問自己：「我能否將自己對\_\_\_\_的貪愛放下？」
7. 然後問自己「榨檸檬」的問題：想象你永遠永遠都得不到\_\_\_\_了。持續釋放，直到你對此百分之百淡定了。
8. 現在問問自己：「我能否將自己對\_\_\_\_的所有貪愛都釋放掉？」重複問自己這個問題，直到你能夠毫不猶豫、毫不抗拒地給出一個肯定的答覆。

---

### 嗔恨——抗拒某物的欲望（範例）

| 我對推銷電話的嗔恨 | 完全釋放了嗎？ |
|---|---|
| 我十分討厭不請自來的推銷。 | |
| 我不喜歡給不認識的人打電話。我永遠都不知道如何正確地介紹自己。 | |

**練習指導：**

1. 寫下自己對某人事物的嗔恨。
2. 寫下自己關於這嗔恨的意識流動。
3. 無論哪種欲望被激發了，釋放它。
4. 寫下自己喜歡它哪一點、討厭它哪一點。
5. 想象自己永遠都不能擁有這個東西了，然後把這一想法激起的感受釋放掉。
6. 問問自己：「我能否將自己對\_\_\_\_的嗔恨放下？」
7. 然後問自己「榨檸檬」的問題：想象你永遠都擁有\_\_\_\_，你永遠都擺脫不了\_\_\_\_。持續釋放，直到你對此百分之百淡定了。
8. 現在問問自己：「我能否將自己對\_\_\_\_的所有嗔恨都釋放掉？」重複問自己這個問題，直到你能夠給出一個百分之百肯定的答覆。

現在我們再來問問自己「榨檸檬」的問題：**想象你永遠都不能\_\_\_\_，永遠、永遠、永遠都不能\_\_\_\_。** 如果這想法引起了一種抗拒感，就垂下頭，讓那股能量上浮離開。**想象你永遠、永遠、永遠都不能\_\_\_\_，** 然後允許那能量浮上表面，允許它穿過你離開——它不好也不壞，只不過是路過的現象。**再問一遍：想象你永遠、永遠、永遠不能\_\_\_\_，** 這會讓你想要被認同、想要控制或是想要安全嗎？不管是哪個「想要」，你能否讓它浮到表面呢？能否把它放下呢？重複釋放，直到沒有相關的無用能量、緊抓感或是抗拒感殘留在你胸腹之間。

現在，我們問自己最後的問題，檢驗一下是否還有能量殘留：**你能否把自己對\_\_\_\_所有的貪愛都放下呢？**

如果你在說「是」的時候感到猶豫放不開，或是很抗拒，就允許那能量上浮，垂下頭，讓它上浮離開。釋放更多，越多越好。

**你能否把自己對\_\_\_\_的所有貪愛都放下呢？** 只要允許那能量穿過你離開。
