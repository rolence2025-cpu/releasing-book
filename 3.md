# 第三章、讓我們鑽研得更深一些

> 愛不僅會帶來全宇宙的力量，還會給你所有的歡愉和智慧。——萊斯特·利文森

現在，我們要開始學習如何釋放、放下，並消除任何干擾我們的東西。

你的腦海中常常會出現一些好的或不好的聲音。問問自己：那是誰的聲音？難道不是你自己的聲音嗎？

既然是你自己的聲音，那一定是你自己設定了它，因此你完全有能力消除它。為什麼要忍受那些把你逼得快瘋掉的聲音呢？明明剛吃完一頓大餐，為什麼沒多久又發現自己站在冰箱前大吃特吃？為什麼你覺得非得罵人不可，即使明知這樣會給自己帶來大麻煩？為什麼開車的時候會變得如此焦躁，甚至開始追趕前面的車子？到底是誰說我們非這樣做不可？

不幸的是，情緒左右著我們的生活。有句話說：「世間所有的亂象，根源都在於情緒的病態。」我們越是容易被情緒牽動，問題就越容易找上門來。所以，讓我們來探索如何消除腦海中那些負面的想法和感受——也就是那些「聲音」。

我們肩膀上頂著的這顆「頭」，其實只是一個錄音兼播放的裝置。它唯一的功能就是把我們經歷的一切記錄下來，再回放出來。如果你要它想想如何賺到一百萬美元，它就會把你所知道的一切關於賺錢的資訊播放一遍。如果你問它自己的健康狀況，它立刻就會把你這些年來累積的所有健康知識全部播出來。然而遺憾的是，這個裝置也會在你不想聽的時候，自動播放那些預先錄好的聲音和想法，干擾你的專注、睡眠和內心的平靜！

## 處理感受的四種方法

一般來說，世界上常見的處理感受的方法有三種：壓抑（隱藏）、表達（宣洩），以及逃避（敷衍）。

在這裡，我們提供你第四種選擇，那就是「聖多納釋放法」，讓你真正釋放感受、化解負面能量。

### 一、壓抑（隱藏）

這是最常見、也是最有害的一種處理方式。把感受硬壓下去，會讓壓抑的能量或壓力不斷累積，最終在你沒有意識到的情況下，驅使你做出一些自己不想要、或頭腦清醒時絕不會去做的行為。長期壓抑感受，最終都會對身心健康造成傷害。

### 二、表達（宣洩）

表達是把感受化為行動，有時候確實能帶來短暫的解脫感。不過，這只是暫時紓解了感受帶來的壓力，並沒有真正把感受消除掉。對被我們發洩情緒的對象來說，這當然不是什麼愉快的經驗。而且，我們事後往往會因為自己的行為感到愧疚，反而讓痛苦加深了。

### 三、逃避（敷衍）

打開電視、進電影院、抽菸、出去玩、聽音樂、喝酒……什麼都行，就是要讓自己遠離那個令人不安的感受。但感受並不會就此消失，它還在那裡，只是往深處沉了一層，在你察覺不到的地方悄悄影響著你。

### 四、釋放

這個方法讓你放下的是感受本身，是處理感受最健康、也是最好的方式。每一次運用這個方法，你就清除了一些積壓已久的負面能量。它讓你一點一點地釋放感受背後的壓抑能量，直到所有的壓抑能量都消散，你會感到越來越輕鬆、越來越平靜。隨著時間推移，你自然而然會變得自由從容，思路也會更加清晰。你的目標和方向會更積極、更有建設性，從而做出更好的決策，工作效率也會跟著提升。
